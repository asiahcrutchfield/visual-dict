# Visual Dictionary

Visual Dictionary is an experimental language-learning project that connects vocabulary directly to images, audio, and simple narrative contexts.

The goal is to make vocabulary acquisition more intuitive by linking words to **visual concepts** instead of translations.

This project is part of a broader exploration into **audio-first language learning** and **comprehensible input for complete beginners**.

---

## Motivation

Traditional dictionaries rely heavily on translation:

貓 = cat

However, this forces the learner to mentally translate between languages.

Visual Dictionary instead connects vocabulary to **concepts**:

image → audio → story context


This approach encourages learners to understand words **directly**, without relying on translation.

---

## Core Idea

Each vocabulary item is represented by three elements:

- **Image** – a clear visual representation of the concept
- **Audio** – native pronunciation
- **Context** – usage inside a simple story

Example:
貓
image: 貓.jpg
audio: 貓.mp3
example: 貓在樹下。


The learner associates the sound and meaning through repeated exposure.

---

## Project Goals

This project explores several ideas:

- Visual vocabulary acquisition
- Audio-first learning
- Controlled vocabulary storytelling
- Comprehensible input for absolute beginners

The long-term goal is to support extremely simple stories that allow learners to understand meaning through **context and repetition** rather than translation.

---

## Relationship to Ground Level Stories

This dictionary is designed to work alongside another project:

**Ground Level Stories**

A set of short narrative episodes using a controlled vocabulary designed for learners with **zero prior knowledge** of the language.

Vocabulary from the visual dictionary is reused in these stories to reinforce learning.
